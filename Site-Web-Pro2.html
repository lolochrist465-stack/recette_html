Site Web pour un restaurant

J'ai réalisé ce projet web personnel dans le domaine de la restauration.

L'objectif ? Permettre à un restaurant de gagner en visibilité et de proposer son menu de pizzas aux utilisateurs.

Ce projet personnel m'a permis de concevoir une interface d'administration sur-mesure, permettant de modifier facilement le menu en ligne — idéal pour un gérant novice dans les nouvelles technologies.

Ce projet simple mais fonctionnel m'a permis de consolider mes compétences en développement web full-stack.

Technologies utilisées : HTML, CSS, DJANGO, PYTHON, HEROKU
